    Baseball API documentation

  • RUWT?!

    The Are You Watching This?! API allows developers to create applications using our integrated sports and television data, as well as our RUWT?! Rating for each game.

    The documentation for the API is at https://areyouwatchingthis.com/api/?baseballhackday

  • Stattleship API

    We make it easy for developers, data scientists, and hobbyists to fetch sports data. api.stattleship.com

    Explore our Developer Documentation and Get Your API Token to see what's possible.

  • SeatGeek

    SeatGeek is a search engine for tickets to live events, including Major League and Minor League baseball games. They have a powerful, easy-to-use API that allows you to tap into their event catalog to obtain information on events, performers, venues, ticket prices and more.

    There is No Rate Limits, No API Keys, and No Caching Restrictions.

    Go to seatgeek.com/build for information on their API , and to platform.seatgeek.com for API documentation
